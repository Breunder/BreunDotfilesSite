Hyprland Dotfiles Configureren

In deze sectie bespreek ik hoe je je dotfiles configureert voor Hyprland.

Stap 1: De basisconfiguratie

De basisconfiguratie van Hyprland bevindt zich in ~/.config/hyprland/hyprland.conf. Hier is een voorbeeld van een eenvoudige configuratie:

# Hyprland Config
exec-once = waybar &
exec-once = Kitty &
bind = SUPER, Return, exec, Kitty
monitor = ,preferred,auto

Deze configuratie start Waybar en Kitty automatisch en stelt een sneltoets in om de terminal te openen met SUPER + Return.

Stap 2: Andere belangrijke instellingen

Je kunt andere belangrijke configuraties toevoegen, zoals:

  • Werkruimtes: Stel specifieke werkruimtes in voor verschillende monitoren.
    • workspace = 1, DP-1
workspace = 2, HDMI-A-1
  • Keybindings: Voeg aangepaste sneltoetsen toe voor je workflow.
    • bind = SUPER, D, exec, rofi -show drun
bind = SUPER, Q, killactive
  • Theming: Pas de kleuren en stijl van Hyprland aan.
    • decoration {
    rounding = 10
    border_size = 2
    active_color = rgba(81, 161, 193, 1)
    inactive_color = rgba(46, 52, 64, 1)
}

Stap 3: Waybar configureren

Waybar is een populaire statusbalk voor Hyprland. De configuratie bevindt zich in ~/.config/waybar/. Hier is een voorbeeld van een eenvoudige configuratie:

{
    "modules-left": ["workspaces"],
    "modules-center": ["clock"],
    "modules-right": ["cpu", "memory", "network"],
    "clock": {
        "format": "{:%H:%M:%S}"
    }
}

Je kunt ook een aangepaste CSS-stijl toevoegen in ~/.config/waybar/style.css om de statusbalk aan te passen.

Stap 4: Rofi configureren

Rofi is een krachtige applicatiestarter. Voeg de volgende regel toe aan je Hyprland-configuratie om Rofi te starten met een sneltoets:

bind = SUPER, D, exec, rofi -show drun

Je kunt de Rofi-configuratie aanpassen in ~/.config/rofi/config.rasi. Hier is een voorbeeld:

configuration {
    font: "JetBrains Mono 12";
    background: "#2e3440";
    foreground: "#d8dee9";
    selected-background: "#81a1c1";
    selected-foreground: "#2e3440";
}

Stap 5: Problemen oplossen

Als je configuratie niet werkt zoals verwacht, controleer dan de logbestanden:

journalctl -xe

Controleer ook of de configuratiebestanden correct zijn gestructureerd en geen syntaxfouten bevatten.
