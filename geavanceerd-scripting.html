Hyprland Geavanceerde Scripting

In deze sectie leer je hoe je geavanceerde scripts kunt maken en integreren in Hyprland.

Stap 1: Scripts toevoegen aan Hyprland

Je kunt scripts toevoegen aan je Hyprland-configuratie door gebruik te maken van de exec-once of exec commando's. Hier is een voorbeeld:

# Start een script bij het opstarten
exec-once = ~/.config/hyprland/scripts/startup.sh

Het script startup.sh kan bijvoorbeeld gebruikt worden om applicaties te starten of instellingen te configureren.

Stap 2: Automatisering met shell-scripts

Shell-scripts zijn krachtig voor het automatiseren van taken. Hier is een voorbeeld van een script dat automatisch werkruimtes toewijst aan monitoren:

#!/bin/bash
# Automatische werkruimte toewijzing
hyprctl dispatch workspace 1 DP-1
hyprctl dispatch workspace 2 HDMI-A-1

Maak dit script uitvoerbaar met chmod +x en voeg het toe aan je Hyprland-configuratie.

Stap 3: Dynamische configuraties

Je kunt dynamische configuraties maken door gebruik te maken van scripts die reageren op systeemgebeurtenissen. Bijvoorbeeld:

#!/bin/bash
# Verander de achtergrond afhankelijk van het tijdstip
HOUR=$(date +%H)
if [ $HOUR -ge 6 ] && [ $HOUR -lt 18 ]; then
    swww img ~/Pictures/wallpapers/day.jpg
else
    swww img ~/Pictures/wallpapers/night.jpg
fi

Dit script verandert de achtergrond afhankelijk van het tijdstip van de dag.

Stap 5: Debugging en logging

Gebruik logging om je scripts te debuggen. Voeg bijvoorbeeld de volgende regel toe aan je script:

#!/bin/bash
# Debugging voorbeeld
echo "Script gestart op $(date)" >> ~/.config/hyprland/scripts/log.txt

Controleer de logbestanden om problemen op te lossen.
