Overzicht

Hier vind je een overzicht van mijn persoonlijke dotfiles configuratie voor Hyprland en andere tools die ik dagelijks gebruik. Deze configuraties zijn zorgvuldig afgestemd op mijn workflow en esthetische voorkeuren.

Belangrijkste componenten

  • Hyprland - Een dynamische tiling Wayland compositor
  • Waybar - Een zeer aanpasbare statusbalk
  • Kitty - Een snelle, feature-rijke terminal emulator
  • Rofi - Een veelzijdige applicatie launcher
  • Swaync - Een lichtgewicht notificatiedaemon met een minimalistische interface
  • Persoonlijke scripts - Voor automatisering en workflow-verbetering

Installatie

Volg deze stappen om mijn dotfiles te installeren op je eigen systeem:

1. Kloon de repository

git clone https://github.com/Breunder/Breundotfiles.git ~/.dotfiles

2. Installeer de dotfiles

./install.sh

Let op: Maak altijd een backup van je bestaande configuraties voordat je nieuwe dotfiles installeert!

# Voorbeeld van mijn Hyprland configuratie
monitor=,preferred,auto,1

# Autostart
exec-once = waybar & hyprpaper & swayncdeamon

# Thema
general {
    gaps_in = 5
    gaps_out = 10
    border_size = 2
    col.active_border = rgba(88c0d0ee)
    col.inactive_border = rgba(595959aa)
    layout = dwindle
}

# Animaties
animations {
    enabled = yes
    bezier = myBezier, 0.05, 0.9, 0.1, 1.05
    animation = windows, 1, 7, myBezier
    animation = windowsOut, 1, 7, default, popin 80%
    animation = border, 1, 10, default
    animation = fade, 1, 7, default
    animation = workspaces, 1, 6, default
}

Waybar

Een minimalistische maar informatieve statusbalk met:

  • Werkruimte-indicators
  • Systeemmonitoring
  • Media controls
  • Netwerk- en batterijstatus

Kitty Terminal

Een snelle, feature-rijke terminal met:

  • Aangepast kleurenschema
  • Ligatuur-ondersteunende fonts
  • Tab- en layoutbeheer

Aanpassingen

Mijn dotfiles zijn ontworpen om gemakkelijk aan te passen aan je eigen voorkeuren:

Waybar modules

Voeg of verwijder modules in ~/.config/waybar/config om de statusbalk aan te passen.

Hyprland configuratie

Pas je Hyprland instellingen aan in ~/.config/hypr/hyprland.conf om je window manager te personaliseren. Hier kun je bijvoorbeeld sneltoetsen, animaties en workspaces configureren.
