Hyprland Dotfiles Basis Scripting

In deze sectie bespreek ik hoe je je scripts configureert voor Hyprland.

1. Systeem Update Script

Dit script houdt je systeem up-to-date door zowel de systeempakketten via pacman als de AUR-pakketten via yay bij te werken:

#!/bin/bash

# Update systeem
sudo pacman -Syu

# Update AUR packages
yay -Syu

Plaats dit script in een bestand genaamd bijvoorbeeld update.sh, maak het uitvoerbaar met:

chmod +x update.sh

En voer het uit met:

./update.sh

3. Starten van Veelgebruikte Programma's

Wil je verschillende applicaties tegelijk starten? Dit script opent bijvoorbeeld je favoriete browser, terminal en muziekspeler:

#!/bin/bash

# Open Firefox
firefox &

# Open Kitty terminal
kitty &

# Open Spotify
spotify &

Pas dit script aan door je favoriete applicaties toe te voegen en voer het uit om ze automatisch te starten.

4. Scherm Resolutie Instellen

Met xrandr kun je de resolutie van je scherm aanpassen. Dit is handig als je meerdere monitors gebruikt:

#!/bin/bash

# Stel de resolutie van HDMI-1 in
xrandr --output HDMI-1 --mode 1920x1080 --rate 60

Pas de uitvoer naar jouw situatie aan (monitornaam en gewenste resolutie).

5. Schakelen tussen Lichte en Donkere Modus

Met dit script kun je snel schakelen tussen een lichte en een donkere modus op je systeem:

#!/bin/bash

# Controleer het huidige thema en wissel tussen licht en donker
current_theme=$(cat ~/.config/hypr/theme)

if [ "$current_theme" == "light" ]; then
    # Zet om naar donker
    echo "dark" > ~/.config/hypr/theme
else
    # Zet om naar licht
    echo "light" > ~/.config/hypr/theme
fi

Dit is een handige manier om je werkruimte snel aan te passen op basis van je voorkeuren voor de dag of nacht.

6. Automatisch Verbinden met een Wi-Fi Netwerk

Met behulp van nmcli kun je automatisch verbinden met een Wi-Fi-netwerk bij het opstarten van je systeem:

#!/bin/bash

# Verbind met Wi-Fi netwerk
nmcli device wifi connect "JouwWiFiNetwerk" password "JouwWachtwoord"

Pas de netwerknaam en het wachtwoord aan naar jouw eigen instellingen.
